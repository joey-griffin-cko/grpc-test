# grpc-test

A safe space to explore grpc with c#

### How to run

Go to `/src/server`
- `dotnet build`
- `dotnet run`

Go to `/src/client`
- `dotnet build`
- `dotnet run`

The server runs on https://localhost:5001 and the client is on https://localhost:5200

There's a `test.http` file with several requests to test and compare functionality.
